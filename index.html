What's inside

Everything you need.
Nothing you don't.

Volex strips away the bloat and gives you a client that stays out of your way.

MR

Modrinth, built in

Browse, search, and install any modpack from Modrinth directly inside the client. No browser tabs, no manual downloads.

JV

Java handled for you

Volex detects the Java version each Minecraft release needs and installs it silently if it's missing.

32x

Parallel downloads

Assets download across 32 parallel threads instead of one at a time. What used to take minutes takes seconds.

GC

Aikar's GC flags

Every instance launches with battle-tested JVM flags that reduce lag spikes and smooth out frame times.

IM

Instance manager

Run multiple versions and modpacks side by side, each with its own RAM allocation and isolated settings.

$_

Live console

Watch game output in real time from inside the client. No digging through log files when something breaks.
